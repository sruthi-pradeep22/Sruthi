BORDER RADIUS

in CSS, the border-radius property is used to create rounded corners for an element's border.

It allows you to control the curvature of the corners by specifying the radius of the circles that form the corners of the element's border box.

The border-radius property can take one to four values, which define the radii of the top-left, top-right, bottom-right, and bottom-left corners respectively.

If you provide only one value, it will be used for all corners.

If you provide two values, the first will be used for the top-left and bottom-right corners, and the second will be used for the top-right and bottom-left corners.
